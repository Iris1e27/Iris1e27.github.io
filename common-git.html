How to push your project to GitHub Repo

  1. Create a repository on GitHub
  2. Open the terminal in the local repository (where the code is located)
  3. git init initialize a local Git repository
  4. git remote add origin [repository URL]
  5. git add . add all the files in the project to the Git repository
  6. git commit -m ‚Äúmessage here‚Äù
  7. git push origin master

How to change remote origin in IDE when you rename repo

  1. Open the terminal in the local repository
  2. git remote -v check the current remote origin
  3. git remote remove origin
  4. git remote add origin [new repository URL]
  5. git remote -v verify that the remote origin has been changed
