如何做一个像这样的网站

Posted on Mon, Jan 9, 2023 教程 项目

如何发布Notion Pages至GitHub

我喜欢Notion，所以我找到了一个博客，它是关于如何构建基于Notion的网站并自动上传到GitHubPages的详细教程。具体请参考博客（链接）或知乎（链接）。

请注意，如果要添加新页面，应复制现有行之一以确保格式一致。

此外，页面的自动上载需要等到 workflow/main.yml 设置的时间。但是，在Notion中完成后，您可以手动执行工作流以等待上载页面，这通常需要不到10分钟的时间。

如果你需要一个留言板…

最后，我添加了一个名为 uttenrenc.es 的GitHub应用程序来启用评论/留言。您可以单击（此处）看看作者是如何解决这个问题的。

或者你也可以把我修改过的 notabolog-starter，然后使用 utteranc.es 更改下面的 notablog-starter/themes/pure-ejs/layouts/post.html 部分，生成自己的脚本。

<%- post.contentHTML %>
    <script src="https://utteranc.es/client.js"
        repo="Iris1e27/Iris1e27.github.io"
        issue-term="pathname"
        label="✨"
        theme="github-light"
        crossorigin="anonymous"
        async>
</script>

你也可以使用 disqus 这是一个收费的强大网络社区平台，适用于各种网站，或使用 giscus，这是一个由 GitHub discussion 提供支持的免费评论系统。

总的来说，disqus 的定价是为了避免广告，但比 giscus 和 utterances 更强大，因为它有许多登录方法和自己的服务器来存储数据（参见此处）。然而，为了免费使用，utterances 和 giscus 应该是更好的选择。
