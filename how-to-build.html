How to publish Notion Pages to GitHub

I love Notion, so I find a blog which is a detailed tutorial about how to build a Notion-based website and automatically upload to GitHub Pages. You may find that the blog’s link or a zhihu’s link is helpful.

Note that if you want to add a new page, you should duplicate one of existing rows to ensure consistent formatting.

Also, automatic uploading of pages needs to wait until the time set by workflow/main.yml. But you can manually execute your workflow to wait for uploading pages after you are done with them in Notion, which usually takes less than 10 minutes.

If you need a Message Board…

Finally, I added a GitHub App called uttenrances to enable comments. You can click here to see how the author solved this issue.

Or you can just fork my modified version of notablog-starter, and then change the post.html part below notablog-starter/themes/pure-ejs/layouts/post.html by using utteranc.es to generate your own scripts. 

<%- post.contentHTML %>
    <script src="https://utteranc.es/client.js"
        repo="Iris1e27/Iris1e27.github.io"
        issue-term="pathname"
        label="✨"
        theme="github-light"
        crossorigin="anonymous"
        async>
</script>

You can also use disqus which is a chargeable powerful networked community platform for your website or giscus which is a free comments system powered by GitHub Discussions to support Message Board in the similar way.

In detail, disqus is priced to avoid ads but more powerful than giscus and utterances, because it has many login methods and its own server to store data (see here). However, for free use, utterances and giscus should be a better choice.
