Welcome !

This is Jingtian’s public website. The purpose of this website is to share my thoughts and ideas.

If you want to build a website like this, please check Steps to build a website like this page.

Why I choose notablog ?

  1. Notion-based. I love generating pages from Notion which has well-designed page style and easy to change format.
  2. Open Source. It is an open source project so I can know the source code and easy to add my own functions if needed.
  3. Free. It doesn’t cost anything.

Last Updated: @2020/09/14

Edited by: @dragonman225

What to do next ?

  1. Write a post on Notion.
  2. Get your website files by executing notablog generate . in your notablog-starter folder.
  3. Upload all files in notablog-starter/public/ folder to your server or any static asset hosting service.
  4. Play with templates and stylesheets in notablog-starter/themes/pure/ folder to customize your blog.
