2023/2/4 CF Zealots Problem Set #41 (Div. 2)
A. 2-3 Moves

先看数据范围10e9，所以肯定用O(logn)的算法，那么想到二分，发现正负不改变结果，2和3的本质区别是一个奇数一个偶数，且多走3步就会时间更少，所以用坐标除以3，分析余数情况0, 1, 2，就可以了。甚至是O(1)复杂度。

import java.lang.reflect.Array;
import java.util.Arrays;
import java.util.HashMap;
import java.util.Scanner;

public class Main{
 
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int t = in.nextInt();
        for(int i=0; i<t; i++) {
            int n = in.nextInt();
            n = Math.abs(n);
            if(n == 1) {
                System.out.println(2); continue;
            }
            // n/3=x...y more 3 better
            int x = n / 3;
            int y = n % 3;
            int res;
            if(y == 0) {
                res = x;
            } else if(y == 1) {
                res = x + 1; //(x-1)+2
            } else {
                res = x + 1;
            }
            System.out.println(res);
        }
    }
}
